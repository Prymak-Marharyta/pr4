# Практична робота №4
Тема: Класи, масиви, ArrayList + GitHub + Maven

У проєкті реалізовано клас Book з конструктором, гетерами/сетерами, методами toString() та equals().  
У драйвер-класі створюється масив з 5 об'єктів Book, частина даних вводиться з клавіатури, після чого вся інформація виводиться у консоль.

Проєкт зібраний за допомогою Maven.
Для перевірки використовується команда:
mvn clean compile
